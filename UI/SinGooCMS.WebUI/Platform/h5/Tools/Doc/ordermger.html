Đơn đặt hàng管理

Đơn đặt hàngTrạng thái有：待付款、待发货、待签收、已完成
待付款：客户创建Đơn đặt hàng成功后，还没有付款，此时Đơn đặt hàng处于“待付款”Trạng thái。
待发货：客户创建Đơn đặt hàng并且已经付款，等待卖家发货，此时Đơn đặt hàng处于“待发货”Trạng thái。
待签收：卖家已经发货，等待客户签收，此时Đơn đặt hàng处于“待签收”Trạng thái。
已完成：客户已经签收了Đơn đặt hàng，Đơn đặt hàng流程完成，此时Đơn đặt hàng处于“已完成”Trạng thái。

客户需要Xử lý的两个Trạng thái有：待付款、待签收

待付款
客户通过第三方支付平台（例如：支付宝、财付通、微信支付）支付Đơn đặt hàng费用，第三方平台收到费用后返回并通知SinGooShop平台更新Đơn đặt hàngTrạng thái，进入Đơn đặt hàng流程Bước tiếp theo。
在待付款Trạng thái，客户可以Lựa chọn 取消Đơn đặt hàng，CloseĐơn đặt hàng将返还Các mặt hàngSố lượng。
超过24小时（Đơn đặt hàng设置中Cấu hình）未付款，Thuộc hệ thống自动取消Đơn đặt hàng并返还库存。

待签收
Đơn đặt hàng发货后，需要签收Đơn đặt hàng。签收Đơn đặt hàng后，Đơn đặt hàng完成。Đơn đặt hàng进入售后Trạng thái。符合售后政策的Đơn đặt hàng（Các mặt hàng）可以发起售后（退换货款等）
卖家发货后，超过7天（Đơn đặt hàng设置中Cấu hình）没有签收（特殊情况需要与卖家协商），Thuộc hệ thống默认签收。

管理员除创建Đơn đặt hàng外，可以全程Xử lý和代Xử lý。
Đơn đặt hàng创建后待付款Trạng thái下，管理员可以Chỉnh sửa 订đơn giá格（如协商需要Chỉnh sửa 价格）。
Đơn đặt hàng付款后，管理员需要发货。发货时需要填写Diễn tả及快递单号。
在客户不方便的情况下，可以委托管理员进行代Xử lý。如线下付款则收款，代签收等。
