常见问题

问：后台管理界面乱了
答：SinGooShop平台UI采用了html5、bootstrap元素，因此请选用IE9（及以上）、火狐（Firefox）、谷歌浏览器（Google Chrome）等支持html5浏览器。

问：无法上传文件
答：1）确保Open了文件上传功能。XemNội dung->Thông số tập tin->Cài đặt upload。如果没有请Open。

2）是否被上传的文件格式与大小所限制

4） 检查网站空间是否磁盘满了！

问：文件上传界面“上传按钮”不显示
答：如图：

上传组件采用flash方式上传，所以浏览器需要支持flash插件，请检查浏览器是否有安装flash插件，并升级到最新的flash版本。

问：无法创建Đơn đặt hàng
答：1）检查Hiện tạiThành viên是否登录
2）在Đơn đặt hàng结算页面时，Thuộc hệ thống将根据Người sử dụngLựa chọn 的送货地址计算运费，要用到Hiện tại网站有效网址，请在“Cấu hình->Cấu hình cơ bản->URL của trang web”中Cấu hình正确的有效的网址。

问：无法向会话Trạng thái服务器发出会话Trạng thái请求，请确保ASP.NET State Service（ASP.NETTrạng thái服务）已启动...
答：没有OpenTrạng thái服务，需要Open服务：
。
如果不Kích hoạtTrạng thái服务，请Chỉnh sửa web.config，去掉Cấu hình<sessionState mode="StateServer" stateNetworkTimeout="20" cookieless="false" timeout="30"/>。

问：SửaBản mẫu时无法保存
答：Thêm、Chỉnh sửa Bản mẫu时需要保存文件到网站本地文件Thuộc hệ thống中，需要可写权限，请Kích hoạtNetwork ServiceNgười sử dụng的可写权限。目录 “/Upload”、“/Templates”都需要可写权限。一般读写Xử lý时报错“Access Denied”都是因为没有可写权限。

问：备份数据库时报错
答：数据库备份是执行“backup”命令，需要数据库有执行此命令的权限，为了网站安全，一般虚拟空间都没有开放此权限！如需要备份数据库请到空间服务商网站手动备份。

问：Thuộc hệ thống邮箱发不出邮件
答：请检查是否正确Cấu hình了Thuộc hệ thống邮箱，请检查Cấu hình“Cấu hình->邮箱服务”。如果Cấu hình没有问题，请检查Dịch vụ emailKích hoạt了SMTP功能。对于批量Gửi e-mail，一般邮箱服务商都进行了限制，需要Xem thêm帮助，请联系邮箱服务商。

问：收SMS非常慢
答：SMS接口没有设置签名、或者没有实名认证（个别SMS商需要），在Gửi SMS时需要审核，造成了SMS发送延时。Xem thêm帮助请联系SMS服务商。

问：以前可以用的支付宝担保交易不能用了
答：支付宝取消了提保交易，只支持“即时到账”。

问：为什么广告显示不出来了
答：1）检查广告Duyệt通过；2）检查广告时效是否过期，这一情况最常发生。

问：我应该申请什么公众号
答：个人申请订阅号。如需要商用，比如需要微信支持等，请申请服务号，申请服务号必须是企业身份。企业号倾向于企业内部使用。
