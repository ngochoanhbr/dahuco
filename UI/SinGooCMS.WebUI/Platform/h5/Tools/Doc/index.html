SinGooShop平台概述

SinGooShop平台基于.Net Framework 4.0、MSSQLSERVER 2005（及以上） 开发，适用于单Người sử dụng网店Thuộc hệ thống搭建。可直接以成品使用，也可以在此基础上进行二次开发。

SinGooShop平台后台管理UI基于Bootstrap 和 Html5，所以浏览器请选用IE9（及以上）、火狐（Firefox）、谷歌浏览器（Google Chrome）。因低端浏览器不支持Htm5，请升级浏览器以达到最佳浏览效果！

SinGooShop平台前端基于NVelocityBản mẫu引擎Thuộc hệ thống，所有前端展示页面均是Bản mẫu生成。Bản mẫu文件存放路径为 “/Templates/”，此路径下每个文件夹就是一个Bản mẫu，在后台Nội dung管理中，设置为Hiện tạiBản mẫuKích hoạt。Người sử dụng也可以自己创建Bản mẫu，放置在此目录下，设置为Hiện tạiBản mẫu即可使用。

SinGooShop电商平台功能强大，Xử lý简单，扩展性好！SinGooShop平台包括Các mặt hàngThuộc hệ thống、Đơn đặt hàngThuộc hệ thống、Thành viênThuộc hệ thống、营销Thuộc hệ thống、以前其它辅助功能。SinGooShop平台支持PC端、手持移动端以及微信端。不同客户端访问会自动转到相应的环境下面，如用Điện thoại di động访问时自动显示Điện thoại di động页面（需要进行相应设置）。如申请了微信公众号，可以在微信端进行Cấu hình，以对接微信平台。

关于二次开发，SinGooShop平台同样全面支持。如果需要新Thêm后台管理页面，请在后台Thuộc hệ thống->菜单管理中添加相应的栏目和模块，Chỉ định thành viên模块文件路径，并ThêmXử lý方式，授权角色权限，拥有此角色的Người sử dụng就能访问这个模块了。

关于前端Bản mẫu语言，SinGooShop平台定义了一系列的功能方法，Người sử dụng只需要按照方法模式书写，就能达到您想要的结果。Bản mẫu语法参见http://doc.sz3w.net。二次开发的Người sử dụng如果需要创建自定义Bản mẫu语言，请在SinGooCMS.BLL.Custom.CMSExt编写相应的功能，前端调用为$cmsext.DoSomeThing(…)。
