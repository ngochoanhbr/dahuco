Nội dung管理

Cấu hình menu
栏目是文章的分类。
每个栏目都必须Chỉ định thành viênTên menu和Logo menu。Logo menu除可以做为索引读取栏目信息外，还用于地址重写。如帮助中心的地址是 /article/Helper
文章的浏览地址是 /article/detail/文章ID。

每个栏目都有Bản mẫuLựa chọn 。

模型管理
Thuộc hệ thống自带有普通文章、图片文章、产品展示、文件下载等4个模型。
普通文章模型：满足一般文章所需要的字段。
图片文章：主要展示图片信息。
产品展示：包含产品相册和产品Giải thích等。
文件下载：包含文件上传字段。
注意：以上Bản mẫu字段都可以增减。

Quản lý bản mẫu
实现对“/Templates/”目录下所有Bản mẫu文件的管理。

有“默认”标识的Bản mẫu是Hiện tại使用的Bản mẫu。如需要使用其它Bản mẫu，点击“设置默认”即可！

如何添加新的Bản mẫu
从官网下载的Bản mẫu或者自己创建的Bản mẫu，把Bản mẫu文件夹拷贝到目录“/Templates/”下。
设置以下参数，点击保存即添加了Bản mẫu。

XóaBản mẫu
XóaBản mẫu并不会物理上把Bản mẫu目录下的文件Xóa，仅仅是逻辑上的Xóa。

文件列表
进入文件列表后，可以Xem/SửaBản mẫu文件。

点击应用保存文件，但不退出Sửa界面。
点击确定保存文件，并且退出Sửa界面。
注意：如果保存文件时报错“拒绝访问”，需要设置Bản mẫu文件夹权限为“可写”。

约定命名
ad文件夹存放的是广告Bản mẫu
inc文件夹存放的是部件文件，前端用 #parse(“inc/header.html”)调用。
user 文件夹存放的是Thành viên功能文件。
mobile 文件夹存放的是Điện thoại di động网站Bản mẫu。
weixin 文件夹存放的是微信网站Bản mẫu。

理论上来说，网站前端都是可以Chỉnh sửa 的，前提是对html、js、css、Bản mẫu语法等有足够的了解。在Chỉnh sửa Bản mẫu之前请做好备份，可在Chỉnh sửa 出错时恢复。
Bản mẫu语法参考 http://doc.sz3w.net。

文件管理
Thông số tập tin
Cấu hình文件是否可以上传，上传文件的格式、缩略图、水印等。
默认文件上传的规则是 \Upload\年\月\随机字符串.扩展名。如：\Upload\2016\3\201603071093120.jpg。
文件上传需要目录\Upload\具有可写权限。

文件夹管理
创建逻辑上的文件夹，用于对上传文件进行归档。
Xóa文件夹并不会对所属文件进行物理Xóa。

在文件管理列表中Xóa的文件，同时会物理Xóa文件。
在Thuộc hệ thống中需要调用的文件都来源于此。
SinGooShop平台所有上传的文件都在这里进行显式管理，大大提高了网站的安全性。
