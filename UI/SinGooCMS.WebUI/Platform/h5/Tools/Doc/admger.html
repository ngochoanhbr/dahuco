Quản lý quảng cáo

Quảng cáo指网页上的某块区域，在这个区域上显示一个或者多个广告。广告形式可以是文字，图片，或者其它如视频等媒体。

广告在前端显示的前提是广告时效未过期且审核通过。

Quảng cáo代码： <script type='text/javascript' src='/include/generatead?placeid=Quảng cáoID'></script>
如 <script type='text/javascript' src='/include/generatead?placeid=2'></script>
