Thuộc hệ thống管理

自定义菜单
如果需要新Thêm模块或者Chỉnh sửa 模块信息，点击Thuộc hệ thống->菜单管理进行相应Xử lý。添加了模块后，应该设置Xử lý。

只有设置了Xử lý的模块，才会出现在头部导航中。

角色管理
角色规定了一组的Xử lý权限。Người sử dụng拥有此角色后，继承此角色的所有权限。超级管理员角色拥有所有模块的权限（模块需要设置有Xử lý权限）。Thuộc hệ thống角色无法Xóa！

勾选Xử lý项，点击保存即可！

账号管理
这里的账号指的是后台管理员账号。 superadmin和admin是Thuộc hệ thống固有的账号，不可Xóa。Superadmin是超级管理员账号，拥有超级管理员角色的所有权限！账号可以同时拥有多个角色的权限！

数据备份

点击相应按钮进行资料备份！备份完成后，请尽快下载并Xóa备份文件，以免资料泄露！ 备份数据库时报错，一般是空间商没有给予权限！请联系空间商解决。

日志Xem
1） 访问日志用于记录所有Người sử dụng的访问请求。需要在Thống kê->Thống kê设置中Open流量Thống kê。对于访问大的网站来说，大量的访问会迅速使数据库爆满，所以建议使用第三方流量Thống kê工具。
2） Thuộc hệ thống日志登录日志记录所有账号登录管理平台的信息。Xử lý日志记录管理员在管理平台进行的Xử lý。
